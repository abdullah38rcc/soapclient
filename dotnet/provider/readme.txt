This web service implementation with C# (.Net) uses:
- Micsosoft Visual C# 2008 Express Edition
- the NUnit-Test-Framework (version 2.4.2 for .Net 2.0)
  See http://www.nunit.org/ for more information.
  
To run the test cases:
1. Compile the solution EpagesProviderWebServices.sln
2. Start the NUnit program "nunit.exe"
3. Load the executable "obj\Release\EpagesProviderWebServices.exe"
4. Click "Run"
